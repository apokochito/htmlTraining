HTML is not a programming language, it's a markup languange

Conditional Comments

Example

<!--[if IE 9]><![endif]-->

<!--[if IE gt 8]><![endif]--> greater than 8

<!--[if IE gte 7]><![endif]--> greater than or equal to 7

<!--[if IE lt 6]><![endif]--> less than 6

<!--[if IE lte 5]><![endif]--> less than or equal to 5

Description

Which are simple HTML comments that IE (up to version 9) uses to identify functional conditions. For example when you need to implement different stylesheets to different versions of browsers.
