Accessible Forms

Example : Labels

HTML Code

        <form>
          <label for="yourName"> Your Name </label>
          <input name="yourName" id="yourName">
        </form>

Description

Each form field should have its own explicit label. The label tag sorts this out, with a for attribute that associates it to a form element.

HTML Code

        <form>
          <fieldset>
            <legend> Name </legend>
            <p> First name <input name="firstName"></p>
            <p> Last name <input name="lastName"></p>
          </fieldset>
          <fieldset>
            <legend> Address </legend>
            <p> Address <textarea name="address"></textarea>
            <p> Postal code <input name="postcode"></p>
          </fieldset>
        </form>

Description

You can group fields using the fieldset tag, and you can set a caption with the legend tag.

Example : Option groups

HTML Code

        <select name="country">
          <optgroup label="Africa">
            <option value="gam"> Gambia </option>
            <option value="mad"> Madagascar </option>
          </optgroup>
          <optgroup label="Europe">
            <option value="fra"> France </option>
            <option value="rus"> Russia </option>
          </optgroup>
        </select>

Description

The optgroup element groups options in a select box.

Example : Navigating fields

HTML Code

        <input name="firstName" accesskey="e" tabindex="1">

Description

Form fields (and field sets) need to be navigated to without the use of a pointing device, such as a mouse.
