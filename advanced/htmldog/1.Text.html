Text: Time, Mark, and “Presentational”

Example

Written by Doctor Who on Thursday 21st November 2052.

<p> Written by Doctor Who on <time datetime="2052-11-21">Thursday 21st November 2052</time>.</p>

Description

If the textual content of the time element is already machine readable, you don’t need the datetime attribute but it is required if it isn’t.

Example

He wants to play with his Legos

<p>He wants to play with his <mark>Legos</mark></p>

Description

This is a form of emphasis, literally speaking, but it won’t always be considered emphasis in the original meaning (for example, the person being quoted above isn’t emphasizing “Legos”, the commenter is).

The line above (<hr> tag) is used to set a thematic break, between pharagraphs.

<i>Text to test 4</i> It is is used for text in an alternate voice or representing a different quality of text.
