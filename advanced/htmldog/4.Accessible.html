Accessible Links

Example : Tabbing

  • Google
  • Facebook
  • Twitter

HTML Code

        <ul>
          <li><a href="https://www.google.com" tabindex="1">Google</a></li>
          <li><a href="https://www.facebook.com" tabindex="3">Facebook</a></li>
          <li><a href="https://www.twitter.com" tabindex="2">Twitter</a></li>
        </ul>

Description

Links that should be in a logical tabbing order, use TAB key to select them, and verify their order to be selected.

Example : Link titles

I'm really bad at writing link text. Click here to find out more.

HTML Code

<p> I'm really bad at writing link text. <a href="inept.html" title="Why I'm rubbish at writing link text: An explanation and an apology."> Click here </a> to find out more. </p>

Description

You can add information to a link using the title attribute to print special text when someone hovers your link.

Example : Accesskeys

Some page

HTML Code

<a href="https://www.google.com" accesskey="l">Some page</a>

Description

You can assign a keyboard shortcut to a link (when the user presses “Alt” or “Ctrl” + the access key)

Example : Skipping HTML

HTML Code

      <header>
        <h1>The Heading</h1>
        <a href="#content">Skip to content</a>
      </header>
      <nav>
         Navigation stuff...
      </nav>
      <section id="content">
        Content...
      </section>

Description

You can supply links that allow users to jump over chunks of your web page.
