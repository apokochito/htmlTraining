HTML5 Forms Pt. 2: Attributes and Data Lists

Example : Placeholder text

HTML Code

        <label for="email_address"> Email address </label>
        <input type="email" placeholder="you@somewhere.com" name="email_address" id="email_address">

Description

It is intended as a hint, rather than a label, for which a label element should still be used.

Example : Autofocus

HTML Code

        <input name="query" autofocus>

Description

When you land on its home page you don't normally need to click on the search box to start typing - you can do so straight away because it already has focus.

Example : Data lists

HTML Code

        <input name="country" list="country_name">
        <datalist id="country_name">
          <option value="Afghanistan">
          <option value="Australia">
          <option value="Austria">
          <option value="Azerbaijan">
        </datalist>

Description

The value of the list attribute in the input element binds it to a datalist element with the corresponding ID.

As a user types in the text field, if what they type matches the start of anything in the data list, those matches will be shown underneath the text field as suggestions.
